목적: uv를 사용하는 Python 워크플로에 대한 간결한 참조다.

권장 워크플로:
- `pyproject.toml`을 정본으로 사용한다
- uv로 의존성을 잠근다
- 임시 전역 설치 대신 uv를 통해 테스트와 도구를 실행한다
- 가상 환경 동작을 명시적으로 유지한다

기록할 내용:
- Python 버전
- 의존성 그룹
- 자주 쓰는 명령
- 테스트 명령
- 포맷팅 및 린팅 명령

규칙:
- 이유를 문서로 남기지 않고 패키지 관리자를 섞지 않는다
- 잠금 파일 변경은 의도적으로만 한다
- 로컬 실행에 필요한 환경 변수가 있으면 문서화한다
