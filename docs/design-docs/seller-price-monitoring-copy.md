# 셀러용 가격 모니터링 화면 문구와 상태 메시지

이 문서는 셀러용 가격 모니터링 서비스의 핵심 화면에서 쓰는 문구, 상태 메시지, 버튼 라벨을 정리한다. 와이어프레임이 정보 구조를 고정한다면, 이 문서는 사용자가 실제로 읽는 텍스트를 고정한다.

## 톤
- 짧고 사실적으로 쓴다.
- 사용자에게 지금 해야 할 행동을 바로 보여준다.
- 경고는 과장하지 말고 원인과 다음 행동을 함께 적는다.
- 내부 용어보다 운영자가 쓰는 용어를 우선한다.

## 1. 랜딩페이지 문구
### 히어로
- 제목: `경쟁사 가격 변동을 놓치지 마세요`
- 부제: `쿠팡·스마트스토어 가격, 쿠폰, 품절, 리뷰 변화를 한 번에 추적합니다`
- 보조 문장: `매일 캡처하고 엑셀 정리하는 운영 대신, 아침 3분으로 오늘 대응할 일을 확인하세요`
- CTA 1: `무료 체험 시작`
- CTA 2: `데모 보기`

### 기능 카드
- `가격 하락 알림`
- `품절/재입고 감지`
- `리뷰 급증 추적`
- `하루 요약 리포트`

### 요금제 문구
- Starter: `작게 시작하는 셀러용`
- Growth: `운영팀 기준으로 확장`
- Agency: `여러 고객사를 함께 관리`

## 2. 대시보드 문구
### 상단 Metric 라벨
- `가격 하락 경쟁사`
- `내 상품보다 싼 경쟁사`
- `마진 위험 상품`
- `품절 전환 경쟁사`
- `리뷰 급증 상품`

### 오늘 바로 대응할 상품
- 카드 제목: `오늘 바로 대응할 상품`
- 빈 상태: `오늘은 대응이 필요한 상품이 없습니다`
- 보조 문장: `가격, 품절, 리뷰 변화가 생기면 여기에서 바로 확인할 수 있습니다`

### 차트 문구
- `최근 7일 최저가 추이`
- `내 기준가와 시장 최저가 비교`
- `이벤트 밀도`

## 3. 상품 상세 문구
### 헤더
- `현재가`
- `최소 마진가`
- `시장 최저가 차이`
- `마지막 수집`

### 테이블 헤더
- `채널`
- `경쟁사`
- `상품명`
- `현재가`
- `배송비`
- `쿠폰 반영가`
- `리뷰 수`
- `평점`
- `품절 여부`
- `수집 시각`

### 액션
- `메모`
- `핀 고정`
- `무시`
- `알림 조정`

## 4. 이벤트 센터 문구
### 필터
- `가격 하락`
- `가격 상승`
- `품절`
- `재입고`
- `리뷰 증가`
- `평점 하락`

### 카드 본문
- `무엇이 바뀌었는가`
- `언제 바뀌었는가`
- `얼마나 바뀌었는가`

### 상태 메시지
- 빈 상태: `아직 감지된 이벤트가 없습니다`
- 보조 문장: `수집이 쌓이면 가격, 품절, 리뷰 변화가 여기에 나타납니다`

## 5. 알림 설정 문구
### 규칙 설명
- `경쟁사 가격이 3% 이상 내려가면 알림`
- `내 최소 마진보다 낮아지면 즉시 알림`
- `품절 전환 시 즉시 알림`
- `하루 1회 오전 8시 요약 메일`

### 보조 문구
- `즉시 알림은 조치가 필요한 변화만 받도록 설정하세요`
- `요약 메일은 아침에 한 번만 정리해서 보냅니다`

## 6. 워크스페이스 / 멤버 / 결제 문구
### 멤버 테이블
- `이름`
- `이메일`
- `역할`
- `상태`
- `초대일`

### 초대 폼
- `이메일 주소`
- `역할 선택`
- `초대 보내기`
- 성공 메시지: `초대 메일을 보냈습니다`

### 플랜 카드
- `현재 플랜`
- `사용량`
- `초과 사용`
- `업그레이드`

## 7. 공통 상태 메시지
### 로딩
- `데이터를 불러오는 중입니다`
- `마지막 성공 수집 시각: {time}`

### 빈 상태
- `아직 등록된 상품이 없습니다`
- `상품 링크를 추가하면 모니터링을 시작합니다`

### 오류 상태
- `수집에 실패했습니다`
- `잠시 후 다시 시도하세요`
- `권한이 없습니다`
- `결제가 만료되었습니다`

### 동기화 지연
- `마지막 수집 이후 변동이 있을 수 있습니다`
- `가장 최근 성공 수집 시각: {time}`

### 부분 실패
- `일부 경쟁상품 수집에 실패했습니다`
- `성공한 항목은 계속 표시합니다`

## 8. 알림 메시지 예시
### 이메일 제목
- `경쟁사 가격이 내려갔습니다`
- `품절 전환이 감지되었습니다`
- `오늘 바로 대응할 상품 3건`

### Slack/웹훅 본문
- `경쟁사 A 가격 8% 하락`
- `내 최소 마진보다 낮아진 상품 1건`
- `재입고 감지 2건`

## 9. 금지할 표현
- `무조건`, `완벽`, `100%`
- 원인 없는 경고 문구
- 내부 수집용 기술 용어만 단독으로 노출하는 표현
