# 설계 문서

설계 문서는 오래 가는 결정을 기록한다. 변경에 이유 있는 트레이드오프가 필요할 때 작성한다.

## 이 섹션의 용도
- 아키텍처 결정
- 핵심 신념
- 재사용 가능한 패턴
- 하나의 계획보다 오래 가야 하는 결정 기록

## 권장 구조
- 문제
- 대안
- 결정
- 트레이드오프
- 결과
- 열린 질문
