# 기획 정책

## 목적
이 디렉터리는 기획팀 전체에 공통으로 적용되는 기준을 둔다. 개인별 Jira 처리 메모나 임시 노트는 두지 않는다.

- [검토 정책](./review-policy.md)
- [인수인계 정책](./handoff-policy.md)

## 사용 원칙
- 정책 문서는 짧고 고정된 규칙만 담는다.
- 예외가 반복되면 개인 메모가 아니라 정책으로 승격한다.
- 정책 변경에는 근거와 적용 범위를 남긴다.
