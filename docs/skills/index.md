# 스킬

## 목적
이 문서는 반복되는 작업 절차를 독립된 단위로 관리한다. 같은 흐름을 여러 작업에서 다시 쓸 수 있을 때만 여기에 둔다.

## 사용 원칙
- 한 스킬은 한 작업 흐름만 다룬다.
- 제품 명세, 설계 문서, 실행 계획을 대체하지 않는다.
- 프로젝트 고유의 일회성 절차는 실행 계획에 두고, 재사용 가능할 때만 스킬로 분리한다.
- 공통 용어는 [용어집](../GLOSSARY.md)과 맞춘다.
- 단계별 검증이 필요하면 [체크리스트](../CHECKLISTS.md)를 함께 갱신한다.

## 권장 구조
- 목적
- 사용 시점
- 전제와 입력
- 절차
- 검증
- 예외와 주의점
- 참고 자료

## 현재 스킬
- [실행 계획 작성](./execution-plan-writing.md)

## 새 스킬을 추가할 때
- 파일명은 짧고 설명적으로 짓는다.
- 이 인덱스에 링크를 추가한다.
- 관련 명세, 설계, 계획 문서와 상호 링크한다.
- 절차가 바뀌면 검증 기준도 같이 갱신한다.
