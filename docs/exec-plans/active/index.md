# 진행 중인 실행 계획

진행 중인 계획은 현재 작업을 살아 있는 문서로 다룬다.

## 규칙
- 하나의 이니셔티브마다 파일 하나를 둔다.
- 계획 파일명은 `YYYY-MM-DD-short-slug.md` 형식을 사용한다.
- 마일스톤이 도착할 때마다 상태를 갱신한다.
- 필요할 때는 검증과 롤백을 포함한다.
- 완료된 계획은 `../completed/`로 옮긴다.

## 템플릿
- 목표
- 범위
- 마일스톤
- 담당자
- 위험
- 검증
- 완료 기준
